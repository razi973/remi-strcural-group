# A Plus Concrete Restoration — Website

A replica of the A Plus Concrete Restoration services page built with plain HTML, CSS, and JavaScript. No frameworks or build tools required.

## Project Structure

```
aplus-concrete/
├── index.html          ← Main HTML page
├── css/
│   └── styles.css      ← All styles (CSS variables, responsive layout)
├── js/
│   └── main.js         ← Smooth scroll, form handler, counter animations
├── assets/             ← Place any local images here
└── README.md
```

## How to Run Locally

Just open `index.html` in any browser — no build step needed.

## How to Deploy (Get a Live URL)

### Option 1 — Netlify Drop (Fastest, free)
1. Go to https://netlify.com/drop
2. Drag the entire `aplus-concrete/` folder onto the page
3. Instant live URL like `https://random-name.netlify.app`

### Option 2 — GitHub Pages (Free)
1. Push this folder to a GitHub repository
2. Go to Settings → Pages → Deploy from main branch
3. Live at `https://yourusername.github.io/repo-name`

### Option 3 — Vercel (Free)
1. Go to https://vercel.com
2. Import your GitHub repo or drag and drop
3. Instant deployment with custom domain support

### Option 4 — Lovable (AI-assisted)
1. Go to https://lovable.dev
2. Start a new project and paste the HTML/CSS/JS
3. Lovable will host it at a `*.lovable.app` URL
